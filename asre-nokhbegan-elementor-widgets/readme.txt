ابزارک‌های اختصاصی المنتور برای وب‌سایت عصر نخبگان (asrenokhbegan.com).

== Description ==

این افزونه دستهٔ اختصاصی «عصر نخبگان» را به ابتدای لیست ابزارک‌های المنتور اضافه می‌کند
و شامل ابزارک‌های زیر است:

* سرتیتر آیکون‌دار — یک آیکون (تصویر/SVG) به‌همراه دو عنوان، با استایل کامل (عادی/هاور).
* لیست عنوان‌ها — لیست فلکس‌باکس از آیتم‌ها با استایل کامل (عادی/هاور).

افزونه از بروزرسانی خودکار از طریق Releaseهای گیت‌هاب پشتیبانی می‌کند.

== Changelog ==

= 1.7.0 =
* افزودن ابزارک «درصد تخفیف» (ووکامرس): نمایش درصد تخفیف برای محصولات دارای فروش ویژه، با انتخاب نوع درصد برای محصولات متغیر (بیشترین / کمترین / درصد قیمت فعلیِ نمایش‌داده‌شده)، پیشوند و پسوند دلخواه، و استایل کامل (تایپوگرافی، رنگ، پس‌زمینه، بوردر، سایه، گردی، حداقل عرض/ارتفاع برای نشان دایره‌ای، چینش، حاشیه و حالت‌های عادی/هاور) به‌همراه استایل مستقل عدد.

= 1.6.0 =
* قیمت محصول: بازطراحی بر پایهٔ «نقش» قیمت‌ها (قیمت فعلی / قیمت پیش از تخفیف) به‌جای فیلدهای خام ووکامرس. اکنون «قیمت فعلی» همیشه با استایل شاخص نمایش داده می‌شود — چه از قیمت ویژه بیاید و چه (در نبودِ تخفیف) از قیمت اصلی؛ یعنی وقتی محصول قیمت ویژه ندارد، قیمت نمایش‌داده‌شده با استایل قیمت فعلی (شاخص) دیده می‌شود.
* قیمت محصول: «قیمت پیش از تخفیف» فقط هنگام فروش ویژه نمایش داده می‌شود (خط‌خورده و به‌صورت پیش‌فرض کم‌رنگ‌تر) و کنترل خط‌خوردگی به آن منتقل شد.
* قیمت محصول: واحد پولِ هر نقش مستقل و مستقیم اعمال می‌شود (منطق گیج‌کنندهٔ نسخهٔ قبلی حذف شد).
* توجه: به‌دلیل تغییر ساختار کنترل‌ها، اگر این ابزارک را قبلاً استایل داده‌اید، لازم است استایل‌ها را یک‌بار دوباره روی بخش‌های «قیمت فعلی» و «قیمت پیش از تخفیف» تنظیم کنید.

= 1.5.2 =
* قیمت محصول: رفع مشکل نبودِ واحد پول وقتی واحدِ قیمت اصلی خاموش است؛ اکنون اگر محصول قیمت ویژه نداشته باشد، قیمت اصلی همان قیمت نهایی است و واحد پول آن همیشه نمایش داده می‌شود (سوییچِ واحدِ قیمت اصلی فقط هنگام وجودِ قیمت ویژه اثر دارد).
* آیکون تکی: بازطراحی کنترل «سایهٔ آیکون» به‌صورت پاپ‌اور تمیز با مقادیر پیش‌فرض نرم و حرفه‌ای (مانند سایهٔ جعبه).

= 1.5.1 =
* قیمت محصول: رفع باگ بی‌اثر بودن «توزیع افقی» (کانتینر اکنون تمام‌عرض است).
* قیمت محصول: آیکون‌های توزیع/تراز بر اساس جهت چیدمان (افقی/عمودی) تغییر می‌کنند.
* قیمت محصول: حذف بخش غیرضروری «کادر کلی».
* قیمت محصول: استایل‌ها وابسته به تنظیمات شدند (کنترل‌های واحد پول فقط وقتی واحد فعال است نمایش داده می‌شوند؛ بخش هر قیمت فقط وقتی آن قیمت فعال است).
* قیمت محصول: خط‌خوردگیِ قیمت اصلی هنگام فروش اکنون یک‌دست روی عدد و واحد پول می‌افتد.
* قیمت محصول: یک «استایل کلی قیمت و واحد» اضافه شد؛ به‌همراه سوییچ «استایل منحصربه‌فرد واحد پول» که در صورت فعال‌شدن امکان استایل جداگانهٔ واحد را می‌دهد.

= 1.5.0 =
* افزودن ابزارک «قیمت محصول» (ووکامرس): نمایش قیمت اصلی و قیمت فروش فوق‌العاده با کنترل کامل فلکس (ترتیب، جهت، تراز، فاصله)، واحد پول مستقل و قابل‌فعال/غیرفعال‌سازی برای هر قیمت با استایل جداگانه، جایگاه فلکس عدد و واحد پول، خط‌خوردگی خودکار قیمت اصلی هنگام فروش، متن «رایگان» برای قیمت صفر، و نمایش کمترین قیمت ممکن برای محصولات متغیر.

= 1.4.0 =
* افزودن ابزارک «آیکون تکی» — فقط یک آیکون (تصویر/SVG) با استایل کامل کادر (پدینگ، حاشیه، پس‌زمینه، بوردر، سایهٔ باکس، گردی گوشه، عرض/ارتفاع) و آیکون (اندازه، شفافیت، رنگ SVG، سایهٔ آیکون از نوع drop-shadow) با حالت‌های عادی/هاور.

= 1.3.1 =

= 1.3.0 =
