# Asre Nokhbegan – Elementor Widgets

ابزارک‌های اختصاصی المنتور برای [asrenokhbegan.com](https://asrenokhbegan.com) — توسعه‌دهنده: **imiladco**.

## ابزارک‌ها

- **سرتیتر آیکون‌دار** (`anw-icon-heading-box`) — یک آیکون (تصویر/SVG، با رندر اینلاین SVG) و دو عنوان؛ استایل کامل آیکون و عناوین با حالت‌های عادی/هاور، فاصله‌ها، تراز و بخش «متن متمایز» برای استایل‌دهی به بخشی از تیتر داخل `<span>`.
- **لیست عنوان‌ها** (`anw-title-list`) — لیست فلکس‌باکس با Repeater؛ کنترل کامل کانتینر (جهت، تراز، wrap، فاصلهٔ افقی/عمودی) و آیتم (تایپوگرافی، پدینگ، گردی، پس‌زمینه، بوردر، سایه) با حالت‌های عادی/هاور.

هر دو ابزارک زیر دستهٔ اختصاصی **«عصر نخبگان»** قرار می‌گیرند که در **ابتدای** لیست ابزارک‌های المنتور نمایش داده می‌شود.

## نصب

1. فایل `asre-nokhbegan-elementor-widgets.zip` را در وردپرس از مسیر **افزونه‌ها → افزودن → بارگذاری افزونه** نصب و فعال کنید.
2. برای آپلود فایل‌های SVG، گزینهٔ **Elementor → Settings → Features → SVG Uploads** را فعال کنید.

## بروزرسانی خودکار (بدون آپلود دستی)

افزونه برای بروزرسانی از **GitHub Releases** پیکربندی شده است. مخزن هدف در فایل اصلی:

```php
define( 'ANW_GITHUB_REPO', 'imiladco/asre-nokhbegan-elementor-widgets' );
```

### انتشار نسخهٔ جدید
1. شمارهٔ نسخه را در هدر `asre-nokhbegan-elementor-widgets.php` و `ANW_VERSION` افزایش دهید (مثلاً `1.0.1`).
2. تغییرات را در گیت‌هاب push کنید.
3. یک **Release** جدید با تگ `v1.0.1` بسازید و ترجیحاً فایل zip افزونه را به‌عنوان asset پیوست کنید (در غیر این صورت از zipball خودکار گیت‌هاب استفاده می‌شود).
4. سایت‌ها ظرف حداکثر ۶ ساعت (یا با کلیک روی «بررسی بروزرسانی») بروزرسانی را نمایش می‌دهند و با یک کلیک نصب می‌شود.

> برای مخزن **خصوصی**، یک توکن در `wp-config.php` تعریف کنید:
> ```php
> define( 'ANW_GITHUB_TOKEN', 'github_pat_xxx' );
> ```

## ساخت فایل ZIP

```bash
bash build.sh   # خروجی: dist/asre-nokhbegan-elementor-widgets.zip
```

## انتشار خودکار روی گیت‌هاب (یک دستور)

اسکریپت `publish.sh` همهٔ مراحل را خودکار می‌کند: ساخت ریپوی Public، push کد،
و ساخت Release با پیوست فایل zip (که بروزرسانی خودکار را فعال می‌کند).

پیش‌نیاز: یک Personal Access Token گیت‌هاب با اسکوپ `repo`.

```bash
GH_TOKEN=ghp_xxxxx bash publish.sh
```

پس از اجرا، ریپو و Release ساخته می‌شوند و کافی است در سایت روی «بررسی بروزرسانی» بزنید.
برای انتشار نسخه‌های بعدی، فقط نسخه را در هدر افزونه بالا ببرید و دوباره اسکریپت را اجرا کنید.
